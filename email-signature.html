Notes

  1. The signature image must be hosted publicly — deploy the site before testing.
  2. Gmail caches images per recipient. Animation plays the first time someone opens the email, then most clients show the final frame (which reads as a clean static signature).
  3. Outlook desktop will likely show only the first frame (start of the stroke). If that's a problem, swap the src to signature.png for an all-static version.
  4. If Gmail strips the image on paste, use the Gmail signature editor's image button and paste the URL https://harrywinteringham.uk/images/signature.gif directly.
